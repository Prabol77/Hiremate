# HireMate Design System (HDS)

Version: 1.0 (Beta)

---

## Philosophy

HireMate follows four design principles:

1. Clarity
2. Consistency
3. Simplicity
4. Professionalism

Every screen should answer:

- What is the most important information?
- What action should the user take next?

---

# Layout

Every page follows the same hierarchy:

Section Header

↓

Metrics (optional)

↓

Main Content

↓

Actions

↓

Footer

---

# Components

## Section Header

Purpose:
Display page title and description.

Used in:

- Dashboard
- Review
- Recommendation
- Interview
- Rewrite
- Cover Letter
- Candidate

---

## Information Badge

Purpose:

Highlight key metadata.

Examples:

🏢 Company

💼 Position

🎯 Match

⭐ Resume Quality

---

## Metric Card

Purpose:

Display KPIs.

Examples:

ATS Score

Resume Match

Missing Skills

Projects

Experience

---

## Document Preview

Purpose:

Render AI-generated documents.

Used for:

- Cover Letter
- Resume Rewrite
- AI Review

---

## Action Bar

Purpose:

Export actions.

Supports:

TXT

DOCX

PDF

Future:

Copy

Share

Print
